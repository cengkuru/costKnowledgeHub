## Key Implementation Models

### 1. Direct Entry
```
Procuring Entity → Transparency Portal → OC4IDS Output
(CoST Honduras, West Lombok)
```

### 2. Extended Procurement System
```
Procuring Entity → Extended Procurement System → OC4IDS + OCDS
(Uganda, Government Procurement Portal)
```

### 3. Hybrid
```
Project Data: Direct → Portal
Contract Data: ← Import from Procurement System
Result: → OC4IDS Publication
```
